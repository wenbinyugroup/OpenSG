Release Notes
=============

.. _OpenSG-1.0:

1.0
-----

1. Homogenization of shell and solid element based 2D SG and 3D SG for beam models. For shell 3D SG, beam stiffness is implemented and requires extensive validation.
2. MSG based classical plate stiffness matrix for shell SG.
3. Dehomogenization of solid cross-section and tapered segment with local buckling analysis.
4. MSG based solid and plate models accessible with working examples. These models are not yet part of the API implementation.
