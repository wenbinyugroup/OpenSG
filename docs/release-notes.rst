Release Notes
=============

.. _OpenSG-1.0:

1.0
-----

1) Homogenization of both and shell solid element based 2D SG and 3D SG are computed for beam model. For shell 3D SG, implemented beam stiffness
 are implemented and requires extensive validation.
2)MSG based classical plate stiffness matrix is used for shell SG.
3) Dehomogenization of solid cross-section and tapered segment is computed with local buckling analysis.
4) MSG based solid and plate models can be accessed using de branch with working examples. However, these working solid models are yet to be the part of API implementation.
